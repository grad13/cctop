# HO-20250626-013: --globalオプション削除による実装変更

**作成日**: 2025年6月26日 19:15  
**作成者**: Architect Agent  
**宛先**: Builder Agent  
**優先度**: Medium  
**カテゴリ**: Implementation Change  

## 📋 変更概要

**--globalオプションをcctopから完全削除**し、ローカル設定（.cctop/）のみの単純な仕様に変更します。

### 背景・理由
- **ユーザー体験の簡素化**: 「どこで実行したか」で設定が決まる予想通りの動作
- **PIL-004との整合性**: デーモンモードでの「プロジェクトディレクトリで実行→そのプロジェクトを監視」のシンプルな体験
- **設定管理の煩雑性回避**: グローバル設定による複雑性を排除

## 🔧 実装変更内容

### 削除対象

#### CLIオプション
```bash
# 削除するオプション
--global
--local  # デフォルト動作になるため不要
```

#### 設定ファイル・ディレクトリ
- **~/.cctop/** ディレクトリの使用停止
- **~/.cctop/config.json** の参照停止

### 変更後の動作

#### シンプルな動作仕様
```bash
# 常に現在ディレクトリの.cctop/を使用
cd /project-a
cctop                    # → /project-a/.cctop/config.json使用

cd /project-b
cctop                    # → /project-b/.cctop/config.json使用
```

#### ファイル管理
```
.cctop/                  # 現在ディレクトリのみ
├── config.json
├── activity.db
├── activity.db-wal     # SQLite自動生成
├── activity.db-shm     # SQLite自動生成  
└── .gitignore
```
