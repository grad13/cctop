# Legacy Code

This directory contains the original cctop implementation (v0.2.x).
It is preserved for reference and gradual migration to the new architecture.

## Structure

- `src/` - Original source code with monolithic architecture
- `shared/` - Original shared module implementation

## Migration Plan

The new architecture (v0.3.x) is located in the `modules/` directory.
Code from this legacy directory will be gradually migrated and refactored
into the new modular structure.
