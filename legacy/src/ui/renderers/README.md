# UI Renderers

This directory contains specialized renderers for different aspects of the CLI display.

## Files

- **EventFormatter.ts** - Formats event data for display (timestamps, paths, numbers)
- **HeaderFooterRenderer.ts** - Renders header and footer sections
- **DetailRenderer.ts** - Renders detailed inspection view (existing)

## Architecture

These renderers handle the presentation logic, separating it from data management and user input handling.
