# MeasurementCalculator.ts
- **行数**: 288行
- **判定**: should
- **理由**: 複数の独立した責務が混在しています。バイナリ検出、行数計算、言語別コード構造解析、統計計算、バッチ処理が同一クラスに集約されており、各責務の独立性が低くメンテナンス性が低下しています。また、複数の箇所でエラーをデフォルト値で握りつぶしており、特にバイナリ判定失敗時にテキストファイルと誤判定される可能性があります。
- **推奨アクション**: 以下のように関心の分離を行うことを推奨します：
  1. **BinaryDetector** - バイナリファイル検出専用クラス
  2. **LineCounter** - 行数計算専用クラス
  3. **StructureAnalyzer** - 言語別コード構造解析専用クラス（Strategy パターン活用）
  4. **MeasurementAggregator** - 統計集計専用クラス
  5. **MeasurementCalculator** - 上記を組合せる Facade クラスに簡略化

  また、エラーハンドリングを改善し、予期しないデフォルト値返却ではなく、適切なエラー報告メカニズムを検討してください。
