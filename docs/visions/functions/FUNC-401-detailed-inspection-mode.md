# FUNC-401: Detailed Inspection Mode

**作成日**: 2025年6月26日 17:05  
**更新日**: 2025年6月26日 18:00  
**作成者**: Architect Agent  
**Version**: 0.2.3.0  
**関連仕様**: FUNC-400, FUNC-402, FUNC-403, FUNC-300

## 概要

選択されたファイルの詳細情報を全画面表示する詳細検査機能。FUNC-402（集約表示）とFUNC-403（履歴表示）の2つのモジュールを統合制御し、詳細表示モード全体の管理を行う。

## 🎯 機能境界

### FUNC-401の責務
- **モード管理**: 詳細表示モードの開始・終了・状態管理
- **モジュール統合**: FUNC-402（上段）とFUNC-403（下段）の統合制御
- **キー入力分散**: FUNC-300からのキー入力をFUNC-403に適切に分散
- **データ統合**: 選択ファイル情報をFUNC-402、FUNC-403に提供

### 他機能との境界
- **FUNC-400**: FUNC-400から選択確定時に呼び出される、戻り時はFUNC-400へ
- **FUNC-402**: 集約表示モジュール - 上段表示の制御・データ提供
- **FUNC-403**: 履歴表示モジュール - 下段表示の制御・キー入力分散
- **FUNC-300**: キー入力管理 - 詳細モード用キー（ESC、q、↑↓）の受信

### 責務外の除外事項
- **具体的表示処理**: 上段・下段の表示処理はFUNC-402、FUNC-403が担当
- **データ取得**: 直接のDB操作はFUNC-402、FUNC-403に委譲
- **ナビゲーション処理**: 履歴内移動の具体的処理はFUNC-403が担当
- **ファイル編集**: 外部エディタ起動等の編集機能は対象外

## 機能仕様

### 基本表示構成

#### 画面レイアウト（例）
```
┌─ File Details ─────────────────────────────┐
│ ← FUNC-402: Aggregate Display Module       │
│ File Name: main.js                         │
│ Total Operations: 15                       │
│ Creates: 1, Modifies: 13, Deletes: 1      │
├─ Operation History (Latest 50) ────────────┤
│ ← FUNC-403: History Display Module         │
│ 2025-06-26 12:34:56 Modified    47     12 │
│ [↑↓] Move  [ESC] Back  [q] Quit  [3/15]  │
└───────────────────────────────────────────┘
```

**詳細仕様**:
- **上段表示**: FUNC-402（集約表示モジュール）を参照
- **下段表示**: FUNC-403（履歴表示モジュール）を参照
- **統合制御**: 本文書（FUNC-401）で説明

### 表示内容

#### 上段表示内容
- **詳細**: FUNC-402（集約表示モジュール）を参照
- **概要**: 基本情報・集約統計の表示

#### 下段表示内容  
- **詳細**: FUNC-403（履歴表示モジュール）を参照
- **概要**: event履歴や統計値履歴の表示

### 操作機能

#### 基本操作
- **ESC**: 詳細モード終了、選択モードに復帰
- **q**: 詳細モード終了（ESCと同じ）
- **↑↓**: 履歴内での移動・フォーカス変更

#### 表示制御
- **履歴表示**: 最新50件の操作履歴を表示
- **フォーカス表示**: 選択中の履歴行を背景色で強調
- **位置表示**: `[現在位置/総件数]` 形式でナビゲーション情報表示

- **モード排他制御**: 他のインタラクティブモードとの同時実行不可

### モジュール制限
