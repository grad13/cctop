## 📝 技術的詳細

### 問題の根本原因
```javascript
// UIState.ts
appendToSearchText(char: string): void {
  this.searchText += char; // 正規化なしで追加
  this.isSearchApplied = false;
}

// UILayoutManager.ts（Dynamic Control Area）
return `{bold}{yellow-fg}Search: [${searchText}${padding}] ...`; // 生のテキスト表示
```

### 推奨される修正
1. `appendToSearchText`で入力文字をフィルタリング
2. または、表示時に`TextNormalizer.getDisplaySafeText()`を使用
