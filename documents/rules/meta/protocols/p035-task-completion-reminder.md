## 🎯 概要

タスク完了を示すキーワードで自動的にチェックリストを提示することで、プロセス実行を確実にし、daily切り出し忘れを防止する。

## 📋 トリガーキーワード

### 日本語トリガー
- 「解決」「完了」「終了」「できました」
- 「修正完了」「実装完了」「対応完了」
- 「✅」「完全解決」

### 英語トリガー
- 「fix:」「feat:」（コミットメッセージ）
- 「completed」「resolved」「done」
- 「finished」「implemented」

## 🔔 自動リマインダー内容

### 標準リマインダー
```
📋 タスク完了チェックリスト確認
documents/rules/meta/checklists/task-completion-checklist.md

特に重要：
- [ ] status/{agent}.md更新
- [ ] dailyへの切り出し
- [ ] 関連ドキュメント更新

すべて完了しましたか？
```

### Git commit前リマインダー
```
🚨 Git commit前の確認
- [ ] status/{agent}.md更新済み？
- [ ] タスク完了チェックリスト確認済み？
- [ ] reports切り出し完了済み？

未完了項目がある場合はcommitを中止してください
```
