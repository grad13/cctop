# Draft Directory

このディレクトリには、正式適用前のドラフト文書を配置します。

## 命名規則

### ドラフト文書
- **形式**: `DRAFT-YYYYMMDD-XXX-title.md`
- **例**: `DRAFT-20250623-001-claude-md-cctop-update.md`

### 分類
- **001-099**: CLAUDE.md更新関連
- **100-199**: 設定・プロトコル更新関連
- **200-299**: プロセス改善関連

### ライフサイクル
1. **Draft作成**: このディレクトリに配置
2. **レビュー・修正**: 必要に応じて修正
3. **適用**: 正式適用後、archives/に移動
4. **削除**: 不要になったドラフトは削除
