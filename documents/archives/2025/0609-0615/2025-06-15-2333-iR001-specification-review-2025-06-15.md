## 3. 重要な仕様確認

### 3.1 データ形式
```
バイナリレコード（9バイト）:
- fileId: 2バイト (uint16) - 最大65,535ファイル
- timestamp: 4バイト (uint32) - Unix時間（秒）
- lines: 2バイト (uint16) - 最大65,535行
- sections: 1バイト (uint8) - 最大255セクション
```

### 3.2 file-mapping.json
```json
{
  "documents/README.md": 1,
  "documents/daily/2025-06-10.md": 61,
  "src/frontend/index.html": 130
}
```

### 3.3 fileMappingの逆引き
```javascript
// 必須：初期化時に逆引きマップを作成
const reverseFileMapping = {};
Object.entries(fileMapping).forEach(([filePath, fileId]) => {
    reverseFileMapping[fileId] = filePath;
});
```
