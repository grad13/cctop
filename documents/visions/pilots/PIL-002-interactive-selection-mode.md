## 概要

リアルタイム監視表示中にキーボード操作で項目を選択し、詳細情報表示や操作実行を可能にするインタラクティブ選択機能。

## 機能仕様

### 基本動作

#### 選択モード開始
- **トリガー**: 監視表示中に方向キー（↑↓）入力
- **初期状態**: 最上位の表示項目を自動選択
- **表示変化**: 選択インジケータ（`>>`）表示開始

#### 選択操作
- **↑キー**: 前の項目を選択
- **↓キー**: 次の項目を選択  
- **Home**: 最初の項目を選択
- **End**: 最後の項目を選択
- **Page Up/Down**: 10項目単位でジャンプ

#### 選択確定・終了
- **Enter**: 詳細モード移行（FUNC-904）
- **Space**: 項目情報のクイックプレビュー表示
- **Esc**: 選択モード終了、通常表示に復帰
- **Tab**: フィルタリング対象として選択項目を設定

### 表示仕様

#### 選択インジケータ
```
>> src/main.js          Modified  12:34:56  2.1KB
   src/utils.js         Created   12:35:10  0.8KB
   test/main.test.js    Deleted   12:35:20  1.5KB
```

#### 選択状態の視覚的表現
- **選択行**: `>>` プレフィックス + 背景色変更（可能な場合）
- **非選択行**: 通常の空白スペース
- **選択範囲外**: 選択可能項目数の表示（例: `[2/15 selected]`）

#### 選択位置の保持
- **画面更新時**: 選択位置を相対的に維持
- **新規項目追加**: 選択位置を自動調整
- **項目削除**: 選択位置を近傍項目に移動

### 高度な選択機能

#### マルチセレクション
- **Shift + ↑↓**: 範囲選択
- **Ctrl + Space**: 個別項目の選択追加/解除
- **Ctrl + A**: 全項目選択
- **選択数表示**: `[3/15 selected]` 形式

#### 絞り込み選択
- **文字入力**: 入力文字に一致する項目のみ表示・選択
- **正規表現**: `/pattern/` 形式での高度なフィルタリング
- **クリア**: Backspace/Delete での絞り込み解除

### 選択可能項目

#### 対象ファイル・ディレクトリ
- **監視中ファイル**: 変更検出されたファイル
- **ディレクトリ**: 監視対象ディレクトリ
- **隠しファイル**: 設定に応じて選択可能

#### 除外項目
- **システムファイル**: .DS_Store等（設定で変更可能）
- **一時ファイル**: *.tmp, *.swp等
- **権限不足ファイル**: 読み取り権限のないファイル

## 技術仕様

### 依存関係
- **FUNC-021**: 二重バッファ描画（選択状態の滑らかな更新）
- **FUNC-022**: CLI表示統合（キーボード入力処理）
- **FUNC-904**: 詳細モード（選択確定時の遷移先）

### キーボード入力処理
- **readline**: Node.js標準ライブラリでの低レベルキー処理
- **非ブロッキング**: リアルタイム監視と並行動作
- **エスケープシーケンス**: 方向キー・特殊キーの適切な認識

### 状態管理
```javascript
// 選択状態の管理構造（実装例）
SelectionState: {
  mode: 'normal' | 'selection' | 'multi',
  selectedIndex: number,
  selectedItems: Set<string>,
  filterPattern: string | null,
  visibleItems: Array<DisplayItem>

```

## 設定項目

### config.json設定
```json
{
  "selection": {
    "enabled": true,
    "multiSelect": true,
    "autoPreview": false,
    "highlightColor": "inverse",
    "pageSize": 10,
    "filterTimeout": 1000
  }
}
```

### 設定項目詳細
- **enabled**: 選択モード機能の有効/無効
- **multiSelect**: マルチセレクション機能の有効/無効  
- **autoPreview**: 選択変更時の自動プレビュー表示
- **highlightColor**: 選択行のハイライト方式
- **pageSize**: Page Up/Down時の移動行数
- **filterTimeout**: 絞り込み入力のタイムアウト（ms）

## 使用方法

### 基本的な使用手順
1. cctop起動後、監視表示を確認
2. ↑または↓キーで選択モード開始
3. 方向キーで目的の項目を選択
4. Enterで詳細モード移行、またはEscで終了

### 高度な使用例
```bash
# マルチセレクション使用例
1. ↓キーで項目選択
2. Ctrl+Spaceで複数項目をマーク
3. Tabでマークした項目のみをフィルタ表示
4. Enterで一括詳細表示

# 絞り込み選択使用例  
1. ↓キーで選択モード開始
2. "test"と入力してtest関連ファイルのみ表示
3. ↑↓で絞り込み結果内を選択
4. Enterで詳細表示
```

## 制限事項

### 技術的制限
- **ターミナル依存**: ターミナルのキーボード処理能力に依存
- **表示領域**: ターミナルサイズによる選択可能項目数制限
- **並行処理**: 大量ファイル変更時の選択応答性低下

### 動作制限
- **最大選択数**: マルチセレクション時の上限（1000項目）
- **フィルタ長**: 絞り込みパターンの最大文字数（100文字）
- **更新頻度**: 高頻度更新時の選択状態リセット
