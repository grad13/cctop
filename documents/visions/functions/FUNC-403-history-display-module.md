# FUNC-403: History Display Module

**作成日**: 2025年6月27日 23:50  
**更新日**: 2025年6月27日 23:50  
**作成者**: Architect Agent  
**Version**: 0.2.3.0  
**関連仕様**: FUNC-000, FUNC-001, FUNC-401, FUNC-402  

## 📊 機能概要

詳細表示モードの下段部分として、選択ファイルのイベント履歴を時系列表示し、履歴内でのナビゲーション機能を提供するモジュール。

**ユーザー価値**: 
- ファイルの変更履歴の完全可視化
- 時系列でのイベント追跡・分析
- 直感的な履歴ナビゲーション

## 🎯 機能境界

### ✅ **実行する**
- イベント履歴の時系列表示・フォーマット制御
- 履歴内での上下移動・フォーカス管理
- 大量履歴の分割表示・スクロール制御
- 現在位置/総件数の表示管理

### ❌ **実行しない**
- 基本情報・統計表示（FUNC-402が担当）
- 詳細モード全体の開始・終了制御（FUNC-401が担当）
- イベントデータの追加・変更・削除
- 履歴の編集・修正機能

## 📋 表示仕様

### 表示レイアウト
```
├─ Event History (Latest 50) ────────────┤
│ Event Timestamp     Event       Lines Blocks │
│ 2025-06-26 12:34:56 Modify      47     12 │ ← フォーカス行（背景色変更）
│ 2025-06-26 12:30:21 Modify.     45     11 │
│ 2025-06-26 12:25:15 Create      32      8 │
│                                           │
│ [↑↓] Move  [ESC] Back  [q] Quit  [3/15]  │
└───────────────────────────────────────────┘
```

### 表示項目仕様

#### 履歴ヘッダー
- **セクション名**: "Event History (Latest 50)"
- **カラムヘッダー**: Event Timestamp / Event / Lines / Blocks
- **境界線**: 罫線による区切り

#### 履歴エントリ
- **Event Timestamp**: イベント発生時刻（YYYY-MM-DD HH:MM:SS形式）
- **Event**: イベントタイプ（Create/Modify/Delete/Move）
- **Lines**: その時点での行数
- **Blocks**: その時点でのブロック数
- **フォーカス表示**: 選択中の行を背景色で強調

#### ステータス表示
- **キー操作ガイド**: `[↑↓] Move  [ESC] Back  [q] Quit`
- **位置表示**: `[現在位置/総件数]` 形式

### データ取得仕様

#### 履歴データ取得
```sql
SELECT 
  e.timestamp,
  et.name as event_type,
  m.line_count,
  m.block_count,
  e.id
FROM events e
JOIN event_types et ON e.event_type_id = et.id
LEFT JOIN measurements m ON e.id = m.event_id
WHERE e.file_id = ?
ORDER BY e.timestamp DESC
LIMIT ? OFFSET ?
```

#### 総件数取得（aggregatesテーブルから）
```sql
SELECT total_events
FROM aggregates
WHERE file_id = ?
```

### ナビゲーション仕様

#### フォーカス制御
- **初期フォーカス**: 最新の履歴エントリ（インデックス0）
- **上移動**: フォーカスを前の履歴エントリに移動
- **下移動**: フォーカスを次の履歴エントリに移動
- **境界処理**: 最上位・最下位での移動制限

#### ページング制御
- **表示件数**: 20件/ページ（設定可能）
- **スクロール**: フォーカスが画面外に出る場合の自動スクロール
- **プリロード**: 次ページの事前読み込み

## 🚀 実装仕様

### 実装要件
1. **HistoryDisplayRenderer クラス**
   - 履歴エントリの表示・フォーマット制御
   - フォーカス行の背景色変更
   - ステータス行の表示管理

2. **HistoryNavigator クラス**
   - 履歴内での上下移動制御
   - フォーカス位置の追跡・更新
   - 境界チェック・スクロール制御

3. **HistoryDataProvider モジュール**
   - eventsテーブルからの履歴データ取得
   - ページング・オフセット計算
   - データキャッシュ・プリロード管理

4. **HistoryFormatter モジュール**
   - タイムスタンプの表示フォーマット
   - 数値の右揃え・パディング処理
   - カラム幅の調整・整列

### フォーカス表示仕様
- **フォーカス行**: 白背景・黒文字（`chalk.bgWhite.black`）
- **通常行**: デフォルト色
- **境界線**: グレー（`chalk.gray`）
- **ヘッダー**: 青背景・白文字（`chalk.bgBlue.white`）

### ページング仕様
- **表示件数**: 20件（設定可能）
- **バッファサイズ**: 前後10件ずつプリロード
- **スクロール閾値**: 上下3行でスクロール発動

## 制限事項

### 技術的制限
- **表示件数制限**: 画面サイズによる表示可能行数制限

### 機能制限
- **単一ファイル**: 複数ファイルの履歴同時表示不可
- **履歴編集不可**: 履歴の削除・修正機能なし
- **リアルタイム更新**: 表示中の履歴自動更新なし
- **検索機能**: 履歴内検索・フィルタ機能なし
