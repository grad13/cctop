# BP-M3: Detail Mode Architecture

**作成日**: 2025年7月7日 22:50  
**更新日**: 2025年7月7日 23:50  
**作成者**: Architect Agent  
**Version**: 2.0.0  

## 📊 概要

**詳細モード**は、streamモード・filterに続く第3段階のマイルストーンで、**ファイル単位でのインタラクティブ詳細表示機能**を提供します。ユーザーが選択したファイルの詳細情報・統計・履歴を総合的に表示し、直感的なナビゲーションを可能にします。

**実装状況**: 仕様策定完了、実装予定（M2 filter完了後）

## 🎯 マイルストーン目標

### インタラクティブ選択の実現
- **ファイル選択**: キーボードによる直感的な単一ファイル選択
- **選択状態管理**: 選択行の視覚的ハイライト表示
- **モード遷移**: 選択→詳細表示のシームレスな切り替え

### 詳細情報表示の実現
- **統合詳細表示**: 基本情報・統計・履歴の統合表示
- **2つの表示方式**: 縦分割（FUNC-401）と左右分割（FUNC-404）
- **履歴ナビゲーション**: イベント履歴内での直感的移動
- **統計可視化**: ファイル単位での集約統計表示

## 📋 対象FUNC仕様（400番台）

### FUNC-400: Interactive Selection Mode
**目標**: リアルタイム監視表示中にキーボード操作で項目を選択し、詳細情報表示や操作実行を可能にするインタラクティブ選択機能

**主要機能**:
- **選択UI管理**: キーボード操作による単一項目選択インターフェース
- **選択状態管理**: 選択状態の追跡・表示制御
- **色表示制御**: 選択行の背景色・文字色変更
- **遷移制御**: 選択確定時の他機能（FUNC-401）への適切な遷移

**状態遷移**:
```
待機状態 ──|上下キー|──→ 選択状態 ──|Enterキー|──→ 詳細状態
(FUNC-202)                (FUNC-400)              (FUNC-401)
    ▲                         │                      │
    │                      |Escキー|                 │
    │                         │                  |Escキー|
    └─────────────────────────┘                      │
           選択終了                                   ▽
                                                 待機状態へ戻る
```

**キーバインド**:
- `↑↓`: 選択移動
- `Enter`: 詳細状態へ移行
- `Esc`: 選択終了

**制限事項**:
- **単一選択**: 1項目のみ選択可能
- **マルチ選択**: 複数項目の同時選択は対象外

### FUNC-401: Detailed Inspection Mode  
**目標**: 選択されたファイルの詳細情報を全画面表示する詳細検査機能。FUNC-402（集約表示）とFUNC-403（履歴表示）の2つのモジュールを統合制御

**主要機能**:
- **モード管理**: 詳細表示モードの開始・終了・状態管理
- **モジュール統合**: FUNC-402（上段）とFUNC-403（下段）の統合制御
- **キー入力分散**: FUNC-300からのキー入力をFUNC-403に適切に分散
- **データ統合**: 選択ファイル情報をFUNC-402、FUNC-403に提供

**表示構成（縦分割）**:
```
┌─ File Details ─────────────────────────────┐
│ ← FUNC-402: Aggregate Display Module       │
│ File Name: main.js                         │
│ Total Operations: 15                       │
│ Creates: 1, Modifies: 13, Deletes: 1      │
├─ Operation History (Latest 50) ────────────┤
│ ← FUNC-403: History Display Module         │
│ 2025-06-26 12:34:56 Modified    47     12 │
│ [↑↓] Move  [ESC] Back  [q] Quit  [3/15]  │
└───────────────────────────────────────────┘
```

**キーバインド**:
- `ESC` / `q`: 詳細モード終了、選択モードに復帰
- `↑↓`: 履歴内での移動・フォーカス変更

**制限事項**:
- **単一ファイル**: 1ファイルのみの詳細表示
- **モード排他制御**: 他のインタラクティブモードとの同時実行不可

---

### FUNC-402: Aggregate Display Module
**目標**: 詳細表示モードの上段部分として、選択ファイルの基本情報と集約統計を表示

**主要機能**:
- **基本情報表示**: ファイル名・パス・最終操作・サイズ情報の表示
- **集約統計表示**: aggregatesテーブルからの統計データ表示
- **静的表示制御**: 上段エリアのレイアウト・フォーマット管理
- **データ取得**: FUNC-000からの基本情報・統計データ取得

**表示レイアウト**:
```
┌─ File Details ─────────────────────────────┐
│ FileID: 123  inode: 456789                 │
│                                            │
│ Created:     2025/06/15 10:30              │
│ Last Update: 2025/06/27 14:22              │
│                                            │
│ Number of Events                           │
│ Create=1  Delete=3   Modify=102            │
│ Move=12   Restore=3  Total=121             │
│                                            │
│ Metric Statistics                          │
│       Byte  Line Block  Date               │
│ First  222   222   444  2025/06/15 10:30  │
│ Last   333   333   555  2025/06/27 14:22  │
│ Max    999   555   777         -          │
│ Avg    456   389   592         -          │
├─ (FUNC-403 境界) ───────────────────────────┤
```

**表示項目**:
- **識別情報**: FileID、inode
- **時系列情報**: Created（最初のイベント）、Last Update（最新）
- **イベント統計**: Create/Delete/Modify/Move/Restore/Total件数
- **メトリック統計**: First/Last/Max/Avgのサイズ・行数・ブロック数

**制限事項**:
- **静的表示**: 表示中のリアルタイム更新不可
- **単一ファイル**: 複数ファイルの同時表示不可

---

### FUNC-403: History Display Module
**目標**: 詳細表示モードの下段部分として、選択ファイルのイベント履歴を時系列表示し、履歴内でのナビゲーション機能を提供

**主要機能**:
- **イベント履歴表示**: 選択ファイルの時系列イベント表示・フォーマット制御
- **履歴ナビゲーション**: 履歴内での上下移動・フォーカス管理
- **ページング制御**: 大量履歴の分割表示・スクロール制御
- **位置表示**: 現在位置/総件数の表示管理

**表示レイアウト**:
```
├─ Event History (Latest 50) ────────────┤
│ Event Timestamp     Event       Lines Blocks │
│ 2025-06-26 12:34:56 Modify      47     12 │ ← フォーカス行（背景色変更）
│ 2025-06-26 12:30:21 Modify      45     11 │
│ 2025-06-26 12:25:15 Create      32      8 │
│                                           │
│ [↑↓] Move  [ESC] Back  [q] Quit  [3/15]  │
└───────────────────────────────────────────┘
```

**表示項目**:
- **Event Timestamp**: イベント発生時刻（YYYY-MM-DD HH:MM:SS形式）
- **Event**: イベントタイプ（Create/Modify/Delete/Move/Restore）
- **Lines**: その時点での行数
- **Blocks**: その時点でのブロック数
- **フォーカス表示**: 選択中の行を背景色で強調

**ナビゲーション**:
- **初期フォーカス**: 最新の履歴エントリ（インデックス0）
- **上下移動**: フォーカスを前後の履歴エントリに移動
- **境界処理**: 最上位・最下位での移動制限
- **ページング**: 20件/ページ（設定可能）

**制限事項**:
- **単一ファイル**: 複数ファイルの履歴同時表示不可
- **履歴編集不可**: 履歴の削除・修正機能なし
- **リアルタイム更新**: 表示中の履歴自動更新なし

### FUNC-404: Dual Pane Detail View
**目標**: 選択されたファイルの詳細情報を左右2ペイン構成で表示する新しい詳細表示モード。FUNC-401の改良版

**主要機能**:
- **左右2ペイン構成**: Event History（左60%）+ File Statistics（右40%）
- **左ペイン**: イベント履歴の時系列表示とナビゲーション機能
- **右ペイン上段**: 基本統計表示（aggregatesテーブル）
- **右ペイン下段**: Advanced統計表示（将来拡張予定）

**画面レイアウト構成**:
```
┌─ Event History ──────────────────────┬─ File Statistics ─────────────────┐
│ Event Timestamp     Event      Lines │ FileID: 123  inode: 456789        │
│ 2025-06-26 12:34:56 Modify      47   │                                   │
│ 2025-06-26 12:30:21 Modify      45   │ Created:     2025/06/15 10:30     │
│ 2025-06-26 12:25:15 Move        32   │ Last Update: 2025/06/27 14:22     │
│ 2025-06-26 12:20:45 Modify      30   │                                   │
│ 2025-06-26 12:15:30 Restore     28   │ Number of Events                  │
│ 2025-06-26 12:10:20 Delete       0   │ Create=1  Delete=3   Modify=102   │
│ 2025-06-26 12:05:15 Modify      25   │ Move=12   Restore=3  Total=121    │
│ 2025-06-26 12:00:10 Create      20   │                                   │
│ ↓ (scrollable)                       │ Metric Statistics                 │
│                                      │       Byte  Line Block            │
│                                      │ First  222   222   444            │
│                                      │ Last   333   333   555            │
│                                      │ Max    999   555   777            │
│                                      │ Avg    456   389   592            │
│                                      │                                   │
│                                      ├───────────────────────────────────┤
│                                      │ Daily Activity                    │
│                                      │ (未実装 - 将来拡張予定)             │
│ [↑↓] Move  [ESC] Back  [q] Quit     │                                   │
└──────────────────────────────────────┴───────────────────────────────────┘
```

**レイアウト仕様**:
- **左ペイン（Event History）**: 60%
- **右ペイン（File Statistics）**: 40%
  - 右ペイン上段（Basic Statistics）: 60%
  - 右ペイン下段（Advanced Statistics）: 40%

**キーバインド**:
- `↑↓`: 履歴リスト内で前後のエントリに移動
- `ESC` / `q`: 詳細モード終了、選択モードに復帰

**FUNC-401との関係**:
- **FUNC-404**: 新しい左右分割設計（推奨）
- **FUNC-401**: 従来の縦分割設計（既存）
- 両方が利用可能、ユーザー設定で選択可能

**制限事項**:
- **画面サイズ依存**: 最小幅80文字、最小高20行必要
- **静的表示**: 表示中のリアルタイム更新不可
- **同時表示制限**: 単一ファイルのみの詳細表示
