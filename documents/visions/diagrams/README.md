# Diagrams ディレクトリ

**作成日**: 2025年6月25日  
**管理者**: Architect Agent

## 📊 概要

技術仕様・設計を視覚化した図表を管理するディレクトリ。Mermaid形式の図表やアーキテクチャ図を格納。

## 📁 ファイル命名規則

```
{prefix}-{index}-{content}.md
```

### プレフィックス
- `DIA` - Diagram（図表）

### インデックス
- 3桁の連番（001から開始）

### 命名例
- `DIA-001-file-lifecycle-state.md` - ファイルライフサイクル状態遷移図
- `DIA-002-system-architecture.md` - システムアーキテクチャ概要図
- `DIA-003-event-flow-mapping.md` - イベントフローマッピング図
