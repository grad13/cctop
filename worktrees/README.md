# Worktree Experiments Repository

**作成日**: 2025年6月30日  
**目的**: Git worktreeを使用した並列開発実験の記録  

## 📋 概要

このリポジトリは、cctopプロジェクトのworktree実験を管理します。v0.3.0.0のDaemon-CLI分離アーキテクチャ開発で使用されるworktree環境の履歴を記録します。

## 🎯 管理対象

- worktree作成・削除の履歴
- 各worktreeでの実験結果
- 並列開発の知見・ベストプラクティス

## ⚠️ 注意事項

- このディレクトリは親git（06-cctop）から独立
- worktree自体の内容はここでは管理しない（各worktreeが独自にgit管理）
- あくまで実験の記録用
